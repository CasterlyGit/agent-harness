03

Why this works

Solo dev, AI augmented. The harness exists for one person who wants the throughput of a team — no daemons, no web app, no permissions model. Markdown is the API. Skills run visibly in a terminal. Every state can be rebuilt from disk on the next invocation.

files are the api

Every interaction crosses a markdown or JSON file boundary. Kill every process and the system reconstructs itself on next read.

visible work

Skills stream in the user's terminal. Read along, interrupt, course-correct. No opaque background pipeline.

fast default

Single-call skills by default. Multi-stage pipelines are opt-in, never silent upgrades. Don't waste tokens.

stateless workers

Each skill invocation re-derives context from disk. Amend cycles are cheap and resumable.
